Everything you need. Nothing you don't.

Designed to do one thing well — open and read files.

⚡

Instant Open

Double-click any file to view it instantly. No vault, no workspace, no project setup required.

🎨

Beautiful Preview

Markdown renders with GFM tables, task lists, and syntax-highlighted code blocks. JSON auto-formats.

📁

Folder Browsing

Open any directory as a workspace. Navigate through files with the sidebar tree view.

🖌

Lightweight Editor

Switch to edit mode for quick changes. Powered by CodeMirror 6 with full syntax support.

🌓

Dark & Light

Follows your system theme automatically. Toggle manually anytime. Both themes are carefully crafted.

📜

Outline Navigation

Auto-generated table of contents from your Markdown headings. Jump to any section instantly.
